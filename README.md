# abeno-portfolio
A portfolio of computer science projects created by Alexandra Beno during the pursuit of a Bachelor's in Computer Science at Western Governors University.

Directory Explanation:
- Computer Science Capstone: A machine-learning powered Video Game Recommendation Service for a fictional company.
    * AlexandraBenoCapstone.ipynb: The main Jupyter Notebook file, created in Python.
    * AlexandraBenoCapstone.pdf: Letter of Transmittal, Proposal Plan, Post-Implementation Report
    * bestSelling_games.csv: Dataset
- Database Management: Several SQL projects relating to database management.
    * Project - Movie Rental Database - SQL: A project involving the manipulation and analysis of a pre-existing database of rentals at a fictional Movie Rental Store.
    * Project - Coffee Shop Database - SQL: A project involving the creation and manipulation of the database used by a fictional Coffee Shop.
- Project - Bike Shop Inventory System - Java: A program providing a user interface to manage the inventory of a fictional bike shop.
- Project - Class Roster System - C++: A program that analyzes makes changes to a dataset of fictional students and their degree programs.
- Project - Delivery Truck Routing - Python: A program instructing a fictional delivery company on how to route their three delivery trucks most efficiently in the process of delivering 40 different packages.
- Technical Communication: A collection of files that relate to proposals, executive summaries, and other documents involved in the communication of projects and ideas at a fictional company.
